Alimentación

Existen dos procesos en la alimentación "Volumen" o "Definición"
Cada uno tiene una utilidad que se explicará acontinuación


Definición


Esta se realiza cuando se quiere bajar el porcentaje de grasa corporal. La desventaje de este proceso

es que cuando se está realizando no se contruye músculo y en ocasiones también se baja el porcentaje

de masa muscular. Personalmente hago 3 comidas al día y lo combino con ayuno intermitente para una mayor

facilidad, puesto que la prioridad es consumir menos calorias de las que se queman en el d[ia


Volumen

En el caso contrario cuando se hace una etapa de volumen es necesario consumir más calorias de las

que se queman, por lo tanto es importante realizar comidas lo más limpias posibles y no procesadas

y además realizar cardio constantemente para preocurar subir la mayor cantidad de masa muscular

sin grasa. Personalmente hago de 5 a 8 comidas diarias.
